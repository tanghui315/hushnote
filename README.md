# HushNote Support

HushNote is a private, local-first recording app for meetings, interviews, and voice notes on iPhone.

If you need help, found a bug, or want to report a problem, please contact:

Email: tanghui315@126.com
Discord: [contact us](https://discord.gg/cdFgTVDdQ4)

## How to get help

When contacting support, please include:
- your device model
- iOS version
- HushNote app version
- a short description of the issue
- steps to reproduce the problem, if possible

## Common topics

### Recording or microphone issues
Please make sure microphone permission is enabled for HushNote in iPhone Settings.

### Transcription or summary issues
If a transcript or summary looks incorrect, include a short description of the session type and what result you expected.

### Privacy
HushNote is designed to keep processing on device by default. If you enable an optional third-party cloud summary provider, transcript content may be sent to the provider you configure.
